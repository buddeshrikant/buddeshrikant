## 🔬 Research

**Bidirectional Neural Machine Translation — An Evaluation-Centric Study**  
*M.Tech Thesis · IIT Palakkad · Supervisor: Dr. Swapnil V. Hingmire*

- Fine-tuned **IndicTrans2 (200M)** with LoRA for English ↔ Hindi and English ↔ Marathi translation
- Designed a novel **LoRA Routing architecture** for NLLB-200-Distilled-600M — direction-specific adapters with deterministic language-based routing, achieving **~66× parameter reduction** vs full fine-tuning
- Built multi-metric evaluation pipelines: BLEU, chrF++, BERTScore, COMET, TER
- Benchmarked on in-domain (Anuvaad) and out-of-domain (IndicMQM) datasets

📄 **Published at WMT Conference (ACL Workshop)** · [ACL Anthology](#)

🤗 **Live Demo (English ↔ Marathi)** · [Try it on Hugging Face Spaces](https://huggingface.co/spaces/shrikantbudde/indictrans-en-mr-translator) — powered by IndicTrans2 (en→mr and mr→en)
